1기능 개요

AI 기획서 작성 기능은 사내 기획자가 웹 브라우저에서 AI를 활용해 기획서 초안을 쉽고 빠르게 작성할 수 있도록 지원하는 기능입니다. 사용자는 기획서 작성에 필요한 정보를 입력하고, AI는 이를 바탕으로 PRD, 기능 명세서, 유저 플로우, 화면 구성 또는 와이어프레임 초안을 생성합니다. 생성된 문서는 사용자가 자유롭게 수정할 수 있어야 합니다.

2배경 및 문제 정의

사내 기획자는 기획서를 작성할 때 단순히 문장을 작성하는 것뿐 아니라, 기획 내용을 구조화하고 구체화하는 데 많은 시간을 사용합니다. 특히 기획 내용을 문서 형태로 정리하는 데 시간이 오래 걸리고, PRD·기능 명세서·유저 플로우·화면 구성 등 여러 산출물을 각각 작성해야 하는 부담이 있습니다.

3목표
  • 기획서 초안 작성 시간 단축
  • 기획 내용의 구체화 지원
  • PRD, 기능 명세서, 유저 플로우, 화면 구성 초안 생성
  • 생성된 문서를 자유롭게 수정할 수 있는 환경 제공
4주요 사용자

주요 사용자는 사내 기획자입니다. 사내 기획자는 업무 과정에서 기획서, PRD, 기능 명세서, 유저 플로우, 화면 구성도 등을 작성해야 하며, 이를 빠르게 초안화하고 수정할 수 있는 도구가 필요합니다.

5사용 환경

웹 브라우저

6핵심 사용자 시나리오
  1. 사내 기획자가 웹 브라우저에서 기획서 작성 기능에 진입합니다.
  2. 사용자는 기획서 작성을 위한 정보를 입력합니다.
  3. AI는 입력된 정보를 바탕으로 기획서 초안을 생성합니다.
  4. 사용자는 생성된 산출물 중 필요한 문서를 확인합니다.
  5. 사용자는 생성된 문서를 자유롭게 수정합니다.
  6. 필요 시 외부 툴 연동을 고려할 수 있습니다.
7생성 산출물
  1. PRD — 제품 또는 기능 요구사항을 정리한 문서
  2. 기능 명세서 — 기능의 동작과 요구사항을 구체화한 문서
  3. 유저 플로우 — 사용자가 기능을 이용하는 흐름을 정리한 문서
  4. 화면 구성 / 와이어프레임 — 화면에 필요한 구성 요소를 표현한 초안
